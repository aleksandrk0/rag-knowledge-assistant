# Векторные хранилища

## Qdrant
Qdrant — векторная база данных, хранящая эмбеддинги вместе с полезной нагрузкой
(payload) и метаданными. Поддерживает фильтрацию по полям одновременно с
поиском ближайших соседей и работает как отдельный сервис по сети.

## HNSW
Приближённый поиск ближайших соседей (ANN) обычно строится на индексе HNSW —
иерархическом графе с навигацией по слоям. Он даёт сублинейный поиск ценой
небольшой потери точности по сравнению с полным перебором.

## pgvector
pgvector — расширение PostgreSQL, добавляющее тип vector и индекс для поиска
по близости прямо в реляционной базе. Удобно, когда не хочется заводить
отдельный сервис и данные уже лежат в Postgres.

## Метрики расстояния
Основные метрики близости: косинус, скалярное произведение (dot product) и
евклидово расстояние. Для нормированных векторов косинус и скалярное
произведение эквивалентны по порядку результатов.
