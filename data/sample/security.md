# Безопасность LLM

## Prompt injection
Промпт-инъекция — атака, при которой вредоносная инструкция во входных данных
подменяет исходную инструкцию системы. Прямая инъекция приходит от пользователя,
косвенная (indirect) — из внешнего контента, который модель читает, например из
веб-страницы или документа в базе знаний.

## OWASP LLM Top-10
OWASP LLM Top-10 — список главных рисков приложений на больших языковых моделях.
На первом месте стоит именно prompt injection; в списке также утечка
чувствительных данных, отравление обучающих данных и небезопасная обработка
вывода модели.

## Jailbreak
Jailbreak — обход встроенных ограничений модели через специально составленный
промпт, заставляющий её выдать запрещённый контент. Защита строится слоями:
фильтрация ввода, системные инструкции и проверка вывода.

## Защита RAG
В RAG особенно опасна косвенная инъекция через проиндексированные документы.
Базовые меры: разделять инструкции и данные, экранировать контекст и не давать
модели исполнять команды, найденные в источниках.
